# Vertex Leaders

AutoLISP для AutoCAD, автоматически создающий:

- координатные выноски по вершинам полилинии;
- номера всех вершин;
- параллельные размеры по каждому сегменту;
- экспорт таблицы координат в Excel и Word.

---

# Возможности

✔ Автоматическая обработка выбранной полилинии

✔ Создание мультивыносок с координатами X и Y

✔ Автоматическая нумерация всех вершин

✔ Построение параллельных размеров по каждому сегменту

✔ Автоматический вынос коротких размеров мультивыноской

✔ Экспорт таблицы координат

- Excel (.xlsx)
- Word (.docx)

# Установка

## 1. Скопируйте файлы

В одну папку необходимо положить:

```
vertex_leaders.lsp
style_template.dwg
```

---

## 2. Откройте AutoCAD

Запустите AutoCAD.

---

## 3. Загрузите LISP

Введите команду

```
APPLOAD
```

После этого выберите

```
vertex_leaders.lsp
```

> **Сюда вставить скриншот окна APPLOAD**

---

## 4. Проверьте загрузку

После успешной загрузки в командной строке появится сообщение

```
vertex_leaders.lsp успешно загружен
```

> **Сюда вставить скриншот командной строки**

---

# Настройка

В коде необходимо указать путь к файлу шаблона.

Найдите строку

```lisp
(setq template-path "C:/...")
```

и замените путь на свой.

Например

```lisp
(setq template-path "D:/LISP/style_template.dwg")
```

# Команды

## VERTEXLEADERS

Основная команда.

Создает

- координаты;
- размеры;
- номера;
- экспорт.

---

## CHECKSTYLE

Проверяет наличие необходимых стилей.

Если стиль отсутствует, автоматически импортирует его из шаблона.

---

# Формат координат

Все координаты имеют вид

```
X = +1234.56

Y = +987.65
```

Всегда используется

- знак +
- знак -
- два знака после запятой

# Возможные ошибки

## Не найден style_template.dwg

Причина

Неверный путь к шаблону.

Решение

Проверьте переменную

```
template-path
```

---

## Не найден стиль "Координаты"

Запустите

```
CHECKSTYLE
```

---

## Команда не запускается

Проверьте, что файл загружен через

```
APPLOAD
```
