COORD_CALLOUT (AutoLISP)

Этот скрипт предназначен для автоматизации процесса простановки координатных выносок на вершинах полилиний в AutoCAD.
Описание

Скрипт считывает вершины выбранной полилинии, создает мультивыноски (MLEADER) с координатами X и Y, а также расставляет номера вершин. Поддерживается экспорт полученных данных в форматы Excel (CSV) и Word (HTML-таблица).
Установка

    Поместите файл vertex_leaders.lsp в любую удобную папку.

    Подготовьте файл-шаблон ++РАМКА++.dwg (если он используется для стилей) и поместите его в ту же папку.

    В AutoCAD загрузите скрипт командой APPLOAD.

Как использовать

    После загрузки введите в командной строке: VERTEXLEADERS

    Выберите полилинию на чертеже.

    Скрипт предложит выбрать режим экспорта:

        Excel — создание файла .csv.

        Word — создание файла .doc (таблица).

        Both — экспорт в оба формата.

        None — экспорт не требуется.

    Дождитесь завершения отрисовки выносок.

Настройка путей

Внутри файла vertex_leaders.lsp необходимо проверить переменные путей, чтобы они соответствовали вашему компьютеру:

    template-path — путь к файлу ++РАМКА++.dwg.

    csv-path — путь для сохранения Excel-файла.

    doc-path — путь для сохранения Word-файла.

Команды

    VERTEXLEADERS — запуск основной функции простановки выносок.

    CHECKSTYLE — вспомогательная команда для проверки наличия необходимых стилей ("Координаты" и "Д-431") в текущем чертеже.

Требования

    AutoCAD (версия с поддержкой AutoLISP/ActiveX).

    Настроенный текстовый стиль «Д-431» (создается скриптом автоматически, если отсутствует).
